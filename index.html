Features

  • Async client with typed APIs for sources, artifacts, notes, chat & research
  • CLI with Click — source, notebook, artifact, label, research commands
  • MCP server adapter (opt-in) for Claude / ChatGPT integration
  • REST API server adapter (opt-in) for HTTP automation
  • Experimental controlled Veo renderer for NotebookLM-authored shots
  • Browser-cookie auth fallback for headless environments

Quick Start

Install:

pip install "notebooklm-py[browser]"

Login:

notebooklm login

List notebooks:

notebooklm notebook list
